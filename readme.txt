# Spreadsheet in C++
A spreadsheet class written in C++ that carries out some Excel-like tasks:

- Each column can be int, double or string using std::variant.
- Rows can be sorted based on a row title
- A union of tables with different row title or column title can be made. 
 
## Build
To run the examples, in a terminal navigate to this directory, then run

mkdir build
cd build
cmake ..
cmake build .
